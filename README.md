# 智慧生產物料預警系統 (Inventory Risk Forecasting System)

# 專案初衷
在訂單管理與生產管理（PC/MC）領域中，傳統 ERP 系統的數據往往過於破碎。本專案旨在演示如何透過數據視覺化，將「靜態的庫存記錄」轉化為「動態的決策支援」，協助管理者提前預判斷料風險。

# 技術架構
- **Frontend:** Chart.js, HTML5, CSS3 (具備響應式設計)
- **Logic:** 實作 ATP (Available To Promise) 預計可用量運算邏輯
- **Deployment:** GitHub Pages

# 核心功能
1. **動態風險預警**：自動計算每日預計庫存，當水位低於 0 時，圖表自動渲染紅色警示區。
2. **情境模擬 (What-If Analysis)**：透過滑桿即時模擬「訂單波動」對庫存水位的衝擊，輔助生管進行排程調整。
3. **資訊降噪**：捨棄冗長的表格，僅呈現關鍵的「趨勢線」與「斷料點」。

# 應用場景
適用於生產管理、物料管理、訂單管理職位，作為 ERP/SAP 系統的加強決策看板。
